Basics of Git & Github

Git and Github are one of the important tools for anyone related to Computer Science field

Basic Command:

git config --global user.name "MtanvirA" -> This initializes the user name.
git config --global user.email "tanviranjum201@gmail.com" -> This recognizes the email address.
git init -> initializes git in the parent folder.
git add filename -> stages the file to be committed.
git commit -m "Message" -> commit the staged file.
git remote add origin repo_address -> adds the Github repo to local git.
git branch -M main -> connects the main branch.
git push -u origin main -> pushes the committed files to the Github.
